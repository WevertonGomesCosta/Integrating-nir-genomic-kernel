1. Introdução

Este script busca dados climáticos diários da API NASA POWER para a localidade de College Station, TX, durante as safras de 2011 e 2012. O objetivo é transformar os dados diários em Covariáveis Ambientais (Environmental Covariates - ECs) que resumem as condições de cada ambiente (ano x manejo hídrico).

Essas ECs serão salvas em um arquivo .csv para serem usadas no script principal de análise (analysis.Rmd) para modelar a interação GxE.

2. Configuração do Ambiente

Carregamos os pacotes necessários para a aquisição e manipulação dos dados.

# Instale se necessário
# install.packages(c("nasapower", "dplyr", "lubridate", "tidyr"))

library(nasapower)
library(dplyr)
library(lubridate)
library(tidyr)

3. Aquisição dos Dados Climáticos Diários

Buscamos os dados para a estação de crescimento (Maio a Setembro) para os anos de 2011 e 2012.

# Coordenadas de College Station, TX
coords_texas <- c(lon = -96.3344, lat = 30.6282)

# Parâmetros climáticos de interesse
params <- c("PRECTOTCORR", "T2M_MAX", "T2M_MIN", "T2M", "RH2M")

# Obtém dados para 2011 e 2012
daily_climate_data <- get_power(
  community = "AG",
  lonlat = coords_texas,
  pars = params,
  dates = c("2011-05-01", "2012-09-30"),
  temporal_api = "DAILY"
) %>% 
  # Converte a data e extrai o ano
  mutate(date = ymd(YYYYMMDD),
         year = year(date))

head(daily_climate_data)

4. Engenharia de Atributos: Criação das Covariáveis Ambientais (ECs)

Agora, agregamos os dados diários para criar variáveis que descrevam cada ano/ambiente.

Definições: * GDD (Growing Degree Days): Usamos a fórmula padrão para milho (Temperatura Base = 10°C, Temperatura Máxima = 30°C). * Stress Hídrico: O número de dias com precipitação zero. * Stress Térmico: O número de dias com temperatura máxima acima de 35°C.

# Calcula GDD diário
daily_climate_data <- daily_climate_data %>%
  mutate(
    T_base = 10,
    T_upper = 30,
    T_eff_min = if_else(T2M_MIN < T_base, T_base, T2M_MIN),
    T_eff_max = if_else(T2M_MAX > T_upper, T_upper, T2M_MAX),
    T_avg_eff = (T_eff_min + T_eff_max) / 2,
    GDD = T_avg_eff - T_base
  )

# Agrega os dados por ano para criar as ECs
environmental_covariates <- daily_climate_data %>%
  group_by(year) %>%
  summarise(
    # Covariáveis de Temperatura
    TMAX_AVG = mean(T2M_MAX, na.rm = TRUE),
    TMIN_AVG = mean(T2M_MIN, na.rm = TRUE),
    GDD_CUM = sum(GDD, na.rm = TRUE),
    HEAT_STRESS_DAYS = sum(T2M_MAX > 35, na.rm = TRUE),
    
    # Covariáveis Hídricas
    PRECTOT = sum(PRECTOTCORR, na.rm = TRUE),
    DRY_DAYS = sum(PRECTOTCORR == 0, na.rm = TRUE),
    RH_AVG = mean(RH2M, na.rm = TRUE),
    .groups = 'drop'
  )

# Replica as linhas para cada combinação de manejo hídrico (WS e WW)
# No seu modelo, a diferença WS/WW já é capturada pelo efeito de ambiente (E)
# As ECs representam o "macro-ambiente" (ano).
ecs_final <- environmental_covariates %>%
  slice(rep(1:n(), each = 2)) %>%
  mutate(water_regime = rep(c("WS", "WW"), times = n() / 2),
         Env = paste0("CS", substr(year, 3, 4), "_", water_regime)) %>%
  select(Env, everything(), -water_regime, -year) # Reordena colunas

# Visualiza as covariáveis finais
print(ecs_final)
